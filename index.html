TecNM Chihuhua II

Programacion web

La programación permite a los desarrolladores crear aplicaciones, sitios web, juegos y soluciones que facilitan la vida cotidiana. Desde las redes sociales hasta los algoritmos que alimentan los motores de búsqueda, todo se basa en código.

Los objetivos de este curso son: Crear un sitio web básico con HTML y hojas de estilo en cascada . Diseñar e implementar una página web dinámica con validación mediante objetos JavaScript. Aplicar diferentes mecanismos de gestión de eventos. Diseñar la página web front-end y conectarla a las bases de datos back-end.
